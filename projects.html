Chess Engine

This chess engine, developed in C++, implements advanced algorithms to play chess. It features a minimax algorithm with alpha-beta pruning for optimal move decision-making. The engine evaluates board positions using a heuristic evaluation function, and it supports various difficulty levels and custom game settings. The project showcases proficiency in data structures, algorithm design, and performance optimization in C++.

GAN for Nebula Images

I've developed a Generative Adversarial Network (GAN) that generates realistic nebula images. This project involved training deep learning models using TensorFlow and Keras. The GAN consists of two neural networks: a generator that creates images and a discriminator that evaluates their authenticity. Through iterative training, the model learns to produce high-quality nebula images. This project highlights the potential of AI in creating stunning visual art and demonstrates advanced knowledge in machine learning and neural network architectures.

Pantry Pal

Pantry Pal is a website for pantry management that also uses AI to generate recipe suggestions. It was built using Next.js, React, Material UI, and Firebase, along with the OpenAI API for AI-based recipe generation. This project helps users manage their pantry inventory efficiently and suggests recipes based on available ingredients, reducing food waste and making meal planning easier.

Key Achievements:
  • Developed a user-friendly interface with Next.js, React, and Material UI.
  • Integrated Firebase for real-time database management and user authentication.
  • Optimized for mobile and desktop views for better user experience.
  • Provided efficient inventory management to minimize food waste.
  • Implemented AI-based recipe suggestions using OpenAI API.
